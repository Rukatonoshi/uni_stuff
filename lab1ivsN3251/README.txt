lab1test
========
Набор тестовых программ для лабораторной № 1 по ОСП. Для сборки - make all.

01. lab1test
------------
Программа для проверки заданной библиотеки на предмет её соответствия требованиям API плагинов.
Пример запуска:
 $ ./lab1test ./libavg.so

02. lab1ivsN3251
------------
Программа для вызова из плагина функции проверки файла. Дополнительно выводит информацию о плагине. Выводит список файлов которые прошли проверки (плагина).
Пример запуска:
./lab1ivsN3251 -P ~/Study/lab1ivsN3251/ -O --pic gif ~/Study/lab1ivsN3251/test/

03. libivsN3251.so
-------------
Сам плагин для 1-й лабораторной работы
Поддерживаемые опции:
 --pic (После, через запятую, указать расширения которые доступны {gif,png,bmp,jpg})
